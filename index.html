Propagation System

GPU-driven propagation with culling, millions of instances at high FPS. Unity + Compute + Forward Lit.

Propagation System - Deep Dive

This system leverages GPU instancing and compute shaders to efficiently render millions of environmental assets. Key features include erosion-aware placement, dynamic noise fields for variation, and advanced frustum and occlusion culling techniques. It supports batched material slots and an artist-friendly layer graph for intuitive control. The system also integrates LOD (Level of Detail) and impostors for performance optimization, along with streaming chunks for large-scale environments.

  • Technology: Unity, C#, Compute Shaders, Shader Graph
  • Features: GPU Instancing, Frustum Culling, Occlusion Culling, LOD, Impostors, Streaming Chunks, Erosion-aware Placement
  • Performance: Achieves high FPS with millions of instances.

Chrono Sphere

Time-warp sphere combining motion vectors + signed distance fields for refraction distortions.

Chrono Sphere - Visual Breakdown

The Chrono Sphere is a complex visual effect that simulates a time-warp distortion. It combines real-time motion vectors with signed distance fields (SDFs) to create dynamic refraction and distortion effects. The core of the effect involves animated Voronoi patterns, curl-noise advection for fluid motion, and a stencil portal mask to define its boundaries. Impact frames are driven by a custom LUT (Look-Up Table) for precise timing and visual punch.

  • Technology: Unity, Shader Graph, VFX Graph
  • Techniques: Motion Vectors, Signed Distance Fields (SDF), Voronoi Noise, Curl Noise Advection, Stencil Masking, LUT-driven Animation
  • Effect: Dynamic time-warp and refraction distortion.

GPU Scene Generator - Tool Overview

This tool provides an artist-friendly, data-driven solution for generating complex scenes entirely on the GPU. It features a graph-based rule system for defining placement logic and modular emitters for various asset types. The system supports on-GPU LOD management and a streaming tilepool for efficient handling of vast environments. It's designed with extensibility in mind, allowing for pluggable shaders and material inheritance to easily customize visual output. Comprehensive documentation is provided for both artists and technical users.

  • Technology: Unity, C#, Compute Shaders, Custom Editor Tools
